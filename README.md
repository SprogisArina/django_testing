# Django testing  

Этот проект представляет собой автоматизированное тестирование двух сервисов:
- Ya_News (новостной сервис с возможностью комментирования)
- Ya_Note (сервис для создания и управления заметками)

## Технологии:
- python 3.9
- pytest 7.1
- Unitest
- Django
